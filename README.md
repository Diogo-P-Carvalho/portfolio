## Getting Started

1. Clone this repo
    ```sh
    git clone https://github.com/Diogo-P-Carvalho/portfolio.git
    ```
2. Install npm packages
    ```sh
    npm install
    ```
3. Run project
    ```sh
    npm start
    ```
